
## Framework

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project description

Create list api for manga website

Node version: 20
Database: Postgress

## Project setup

```bash
$ yarn install
```

## Compile and run the project

Create env.dev / env.stg / env.prd
# Choose project env
```bash
export PROJECT_ENV=dev
```

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
